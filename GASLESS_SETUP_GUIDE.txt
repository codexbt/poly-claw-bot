WHAT IS GASLESS TRADING?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Normal Polymarket Trading:
  ❌ Need MATIC gas (gas fees on Polygon)
  ❌ Each transaction costs gas
  ❌ Slower execution

Gasless Trading (with Relayer API):
  ✓ NO MATIC gas needed
  ✓ Relayer abstracts gas fees
  ✓ Transactions signed locally, relayer executes
  ✓ Much faster & cheaper
  ✓ Perfect for high-frequency trading (1-second intervals!)

HOW THE GASLESS FLOW WORKS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Every Trade:

  1. Signal Generated
     • Momentum + candle analysis triggers trade
     • Signal score calculated
     
  2. Attempt Gasless (RELAYER API):
     • Order sent to: https://relayer.polymarket.com/orders
     • Headers: X-API-KEY: YOUR_RELAYER_API_KEY
     • Payload: tokenId, amount, side (BUY), orderType (FOK)
     • Response: orderID (GASLESS transaction!)
     
  3. If Gasless Fails → FALLBACK to Standard CLOB:
     • Use regular CLOB client
     • May require MATIC gas
     
  4. Order Tracking:
     • Log transaction type: [GASLESS] or [CLOB]
     • Track entry price & signal direction
     • Monitor for reversals

Solution: 
  • Relayer API might be rate-limited
  • Check Polymarket relayer status: https://relayer.polymarket.com/account
  • CLOB fallback will work fine (may need MATIC for gas)

Problem: "Order submission FAILED"
Solution:
  • Check wallet has USDC on Polygon
  • Verify POLYMARKET_FUNDER_ADDRESS has USDC balance
  • Check .env credentials are correct

Problem: "Authentication failed" error
Solution:
  • Verify RELAYER_API_KEY is correct in .env
  • Your key: YOUR_RELAYER_API_KEY ✓
  • Check key is active on Polymarket profile
