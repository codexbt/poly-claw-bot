# Poly Claw Bot

This repository contains the Poly Claw Bot trading system and related Python automation scripts.

## Overview

- Automated trading and backtesting tools for cryptocurrency and sports markets
- Includes bot scripts, market discovery tools, diagnostics, and helper utilities
- Uses Python and optional Node/JS support for integrations

## Getting Started

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Create a `.env` file from the provided example and set required credentials.

3. Run the desired bot script, for example:

```bash
python bot.py
```
